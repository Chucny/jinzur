# jinzur
Jinzur, is a remote controlling malware that installs itself inside computers when running the python file for the first time and lets the hacker control the PCs remotely.
<br>
Run the Jinzur-malware on a Linux/Windows PC to infect it. Change the IP to your computer's IP address.<br>Dependencies: threading, os, socket<br>
For this to work, forward TCP Port 5555 on your wifi router's settings. Then, you can control the computers that are infected.
